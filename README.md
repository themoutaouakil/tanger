# Tanger Website

Website for IMIGO Immigration - Tanger office.

Built with Next.js 14, React 18, TypeScript, and Tailwind CSS.

## Features

- 5 pages: Home, Services, Processus, Blog, Contact
- Responsive design (mobile-first)
- SEO optimized
- Contact form
- Modern UI/UX

## Local Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
npm start
```

## Deployment

This repository is configured for Dokploy deployment.

### Dokploy Settings

```
Type: Docker
Dockerfile: Dockerfile
Port: 3000
```

No Build Path needed - all files are at root.
